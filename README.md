 Watcher-REST-APIThe Central Nervous System of the Watcher EcosystemWatcher-REST-API ek robust backbone hai jo user settings, database persistence, aur real-time alerts ko handle karta hai. Ye system sirf data store nahi karta, balki ye decide karta hai ki kab user ko notify karna hai aur kab AI engine (FastAPI) ko task assign karna hai.🏗️ The Watcher Ecosystem Flow (System Design)Code snippetgraph TD
    A[Chrome Extension] -->|Captures URL| B(Java Spring Boot API)
    B -->|Persists Data| C[(PostgreSQL)]
    B -->|Sends Data for Analysis| D[FastAPI AI Engine]
    D -->|Change Detected: True/False| B
    B -->|Trigger Alert| E{Notification Engine}
    E -->|SMTP| F[Email Sent to User]
    E -.->|Upcoming| G[SMS / Twilio]
🌟 Key Technical Highlights1. Reliable Persistence (PostgreSQL)Humne PostgreSQL ka use kiya hai user-provided URLs ki history, monitoring frequency, aur state changes ko track karne ke liye. Isse data integrity aur complex queries (like tracking trends over time) handle karna easy ho jata hai.2. Notification Pipeline (Java Mail Sender)Jab Watcher-FAST-API content mein shift detect karta hai, ye API automatically trigger hoti hai.Email Body: Isme purana data vs naya data ka difference aur change hone ka timestamp user ko bheja jata hai.Security: Mail credentials ko environment variables ke through secure rakha gaya hai.3. Inter-Service CommunicationJava aur Python ke beech seamless communication ke liye humne efficient REST calls ka use kiya hai, jisse system decoupled rehta hai (Microservices-ready).🛠️ Deep Tech StackCore: Java 21 & Spring Boot 3.xORM: Spring Data JPASecurity: BCrypt for password hashing (if applicable)Database: PostgreSQL 15+External Integration:JavaMailSender for SMTPWebClient for Async API calls to FastAPI📡 Essential Endpoints (Preview)MethodEndpointDescriptionPOST/api/v1/watchersNaya URL track karne ke liye add karein.GET/api/v1/watchers/statusCurrent active monitoring tasks ki list.PATCH/api/v1/watchers/{id}Tracking frequency ya notifications update karein.POST/api/v1/notify(Internal) FastAPI call karta hai change report karne ke liye.🚧 Upcoming Enhancements (Roadmap)[ ] SMS Integration: Twilio ya Fast2SMS ke through instant mobile alerts.[ ] Custom Sensitivity: User set kar sakega ki use "Minor" changes pe mail chahiye ya sirf "Major" updates pe.[ ] OAuth2 Social Login: Google/GitHub se extension login.🛡️ LicenseDistributed under the MIT License.
